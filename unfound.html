Summer Internship at UnFound
21 May 2018 - 14 July 2018

In March 2018, I signed up on AngelList to find myself a front-end web development internship. I applied to UnFound because the idea of killing misinformation and giving people a holistic perspective on current affairs intrigued me. It was a cause I wanted to be a part of. As my assignment, I was provided a website and asked to replicate it using VueJS or Angular4. I chose to develop with VueJS since I had never worked with a framework before and VueJS has a smoother learning curve than Angular4. I created an almost exact replica of the provided website and was offered the internship.

I interned with UnFound for 7 weeks. I was tasked with building the front end of their web app from ground up in React. Having no prior experience in React and no other member on the team to help me out with it, I started familiarising myself with the framework and experimented with it for the first few days.

I was provided with low fidelity mockups of the web app. I took the following steps to build the application from scratch:

  1. Created reusable UI components
  2. Created page layouts and rendered the UI components with dummy data
  3. Set up routing with react-router
  4. Fetched real data from UnFound's APIs
  5. Configured authentication using Firebase
  6. Set up basic global state management using Redux
Throughout this process, I used Bitbucket as directed and saw how version control was implemented in a professional environment.

I completed my internship by handing over a working MVP to UnFound. It was the first time I was able to participate in the professional development process of a product. I observed how they designed the architecture, made techinal design decisions and debugged their code. I saw immense growth in my technical know-how and realised how important it is to surround myself with people more skilled, knowledgable and experienced than me.
